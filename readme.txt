= Key Features =  
- **Automatic Conversion:** Seamlessly converts JPEG and PNG images to WebP format **upon upload**, ensuring your media library stays optimized.  
- **Quality Settings:** Customize WebP image quality with adjustable settings, giving you complete control over the balance between image quality and file size.  
- **Bulk Conversion Tool:** Convert existing images in your library to WebP in just a few clicks.  
- **Unique File Names:** Ensures that converted WebP files have unique names to avoid overwriting issues.  
- **User-Friendly Interface:** Simple, intuitive settings panel for effortless image management.  
- **Performance Boost:** Dramatically improve page load speeds with optimized images.  
- **Compatibility:** Works with all WordPress themes and integrates smoothly with popular image optimization plugins.  

= Why Choose Negarara? =  
Whether you're a blogger, e-commerce site owner, or developer, **Negarara** ensures faster page loads and optimized visuals, enhancing both user experience and SEO rankings.  

== Installation ==  
1. Upload the plugin files to the `/wp-content/plugins/negarara` directory, or install the plugin directly through the WordPress Plugins screen.  
2. Activate the plugin via the **Plugins** screen in WordPress.  
3. Adjust settings under the **Media** menu, and upload images to start converting them to WebP.
